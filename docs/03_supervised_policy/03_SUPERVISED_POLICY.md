# 03_SUPERVISED_POLICY

## Goal

Train a strong supervised base policy for Japanese riichi mahjong decisions.

## Scope

- Discard decisions.
- Riichi decisions.
- Call decisions.
- Kan / win / pass decisions.
- Defense and fold behavior.
- Rank-aware late-game behavior.

## Rule

Supervised accuracy is only a proxy. A model is not promoted unless evaluation metrics improve.
