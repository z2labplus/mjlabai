## Primary metrics

| Metric | Why it matters | Target | Status |
|---|---|---:|---|
| Stable dan estimate | Direct north-star metric | > 10.68 | Not measured |
| Tenhou pt EV | Aligns with rank progression | Positive vs baseline | Not measured |
| Average rank | General strength | Lower is better | Not measured |
| Fourth-place rate | Tenhou punishes fourth heavily | Lower is better | Not measured |

## Reporting rule

Every experiment report must include primary metrics when available. If unavailable, it must explain why and state which proxy metric was used.
