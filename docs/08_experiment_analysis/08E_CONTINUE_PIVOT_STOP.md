# 08E_CONTINUE_PIVOT_STOP

## Continue if

- Core metrics improve with acceptable uncertainty.
- Fourth-place rate does not regress dangerously.
- Failure cases are explainable and fixable.

## Pivot if

- Loss improves but Tenhou-oriented metrics do not.
- Evaluation shows systematic strategic weakness.
- Data or environment mismatch is the likely bottleneck.

## Stop / pause if

- Evaluation is invalid.
- Compliance risk is unresolved.
- Current path cannot plausibly improve stable dan.
