# 04B_REWARD_DESIGN

| Reward component | Purpose | Weight | Risk |
|---|---|---:|---|
| Final rank / pt | Align with Tenhou progression | | Sparse reward |
| Fourth-place penalty | Avoid Tenhou rank damage | | Over-defensive play |
| Hand result | Learning signal | | May conflict with rank EV |
| Deal-in penalty | Defense learning | | Over-folding |

## Rule

Reward changes require evaluation against average rank, pt EV and fourth-place rate.
